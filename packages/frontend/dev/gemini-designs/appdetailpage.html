Weather Station Deluxe

By DevGuru42

Published: Jan 15, 2024 | Last Updated: May 20, 2024

View Source
ESP32 IoT Sensors

Description

This comprehensive weather station application for the ESP32 microcontroller accurately measures and displays real-time temperature, humidity, and atmospheric pressure. It leverages the BME280 sensor for precise readings.

Key features include:

  • Data logging to an SD card.
  • Wireless data transmission to a cloud platform (e.g., ThingSpeak, MQTT broker) for remote monitoring.
  • A user-friendly web interface accessible via the ESP32's Wi-Fi access point for local data viewing and configuration.
  • Optional OLED display support for direct visual feedback.

The project is well-documented and includes example code for easy setup and customization. It's perfect for hobbyists, students, or anyone interested in building their own environmental monitoring system.

// main.ino - Weather Station Deluxe
#include <WiFi.h>
#include <Wire.h>
#include <Adafruit_Sensor.h>
#include <Adafruit_BME280.h>

#define SEALEVELPRESSURE_HPA (1013.25)

Adafruit_BME280 bme; // I2C

void setup() {
  Serial.begin(115200);
  Serial.println(F("BME280 Test"));

  bool status = bme.begin(0x76);  // Check your BME280 I2C address
  if (!status) {
    Serial.println("Could not find a valid BME280 sensor, check wiring!");
    while (1);
  }
  // ... more setup code
}

void loop() {
  Serial.print("Temperature = ");
  Serial.print(bme.readTemperature());
  Serial.println(" *C");

  Serial.print("Pressure = ");
  Serial.print(bme.readPressure() / 100.0F);
  Serial.println(" hPa");

  Serial.print("Humidity = ");
  Serial.print(bme.readHumidity());
  Serial.println(" %");

  Serial.println();
  delay(2000);
}
